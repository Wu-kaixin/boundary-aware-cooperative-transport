## Asset Policy

The README files now reference only Git-tracked assets under `docs/assets/`.

Generated files under `runs/` and `platforms/mas_public/data/` are useful for local experiments, but they are ignored by Git. If a figure, GIF, or video should appear on GitHub, copy it into `docs/assets/`, give it a stable descriptive name, and reference that path from the README.
